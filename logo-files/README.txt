MAIN BRANCH CAPITAL — logo files
================================

Mark: genesis block + main branch (bold, copper terminal block) + orphaned fork (faded).

Colors
  Deep navy   #0F2540
  Ink/charcoal #14181E
  Copper      #D08A3E
  Fork grey (on light) #B9BEC6
  Fork grey (on navy)  #54637A

Typography
  Archivo (Google Fonts) — wordmark weight 800, caps line weight 600, letter-spacing ~0.34em.
  SVG lockups use live <text>; install Archivo before importing into design tools,
  or use the PNG versions.

  lockup-stacked-*.svg ................. badge + wordmark, stacked
  png/ ................................. 1024px raster versions of marks & badges
